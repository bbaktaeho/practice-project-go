# search-word-go

여러 파일에서 원하는 단어를 찾는 프로그램입니다.

### Prerequisites

- go 1.18

### Build

```
go build
```

### Usage

```
./search-word-go <단어> <파일1> <파일2> <파일3> ...
```

### Example

- 프로그램 실행

  ```
  ./search-word-go 혜택 test-files/*
  ```

- 결과

  ```
  찾으려는 단어: 혜택

  test-files/wanted.txt
  -----------------------------------------------------
          18      혜택 및 복지
          31      ◈ 혜택 및 근무환경
  -----------------------------------------------------

  test-files/wanted2.txt
  -----------------------------------------------------
          23      혜택 및 복지
  ```
